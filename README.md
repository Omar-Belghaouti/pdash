# pdash
orders dashboard in microservices architecture

![image](https://user-images.githubusercontent.com/48713070/187465539-cd64ef2c-9876-49a1-b89e-dd9d8062562e.png)

## run with
```sh
docker-compose up --build -d
```

## stop with
```sh
docker-compose down
```

## versions
- docker: `20.10.17`
- docker-compose: `1.29.2`
